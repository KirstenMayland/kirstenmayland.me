Orienteering

Map + Compass + Running

Orienteering is an adventure running sport where you try to navigate from checkpoint to checkpoint using a compass and terrain features on a map. The checkpoints are called 'controls' (see right) and you carry an 'E-punch' that will record the time you get to each control (fastest total time wins). This allows competitors to start minutes in order to prevent following.
